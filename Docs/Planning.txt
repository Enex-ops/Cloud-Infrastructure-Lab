# GT3 Cloud Architecture (Free-Tier Version)

---

## Edge / Global Services

- **Client**  
  ↓

- **Route 53** → DNS routing for your domain  
  _Cost: ~$0.50/month per hosted zone (minimal)_  
  ↓

- **CloudFront** → CDN for static and dynamic content  
  _Free Tier: 1TB data transfer out, 50GB storage per month_  
  ↓

- **S3** → Static assets storage, served via CloudFront  
  _Free Tier: 5GB storage, 20k GET/PUT requests/month_  

- **DynamoDB** → NoSQL database for application data  
  _Free Tier: 25GB storage, 25 RCU/25 WCU_  

- **IAM** → Roles, policies, and permissions  
  _Free Tier: 100% free_

---

## VPC / Networking

- **VPC** → Main network for private and public resources  

### Public Subnets
- **Internet Gateway** → Connects public subnets to the Internet  

### Private Subnets
- **Lambda Functions** → Application logic, may access DynamoDB/S3  

- **Security Groups** → Firewalls controlling inbound/outbound traffic  

_Notes: Avoid NAT Gateway to stay free-tier safe_

---

## Compute & Application Flow

1. Client requests → Route 53 → CloudFront → API Gateway  

2. API Gateway triggers Lambda functions  

3. Lambda functions interact with:  
   - DynamoDB (database)  
   - **Optional:** ElastiCache (skip for free-tier)  

4. Responses flow back → API Gateway → CloudFront → Client  

---

## Monitoring & Observability

- **CloudWatch** → Logs, metrics, and alarms for Lambda and API Gateway  
  _Free Tier: 5GB logs/month, 10 custom metrics_

---
